Installation¶

This project is not yet released on PyPi.

You can install the project manually by cloning the repo, and using the included Makefile.

git clone https://github.com/rbpatt2019/studious-happiness/
make install

If you would like to contribute to development, the install instructions are slightly different. Please see the section on For contributors.
